## Requirements :
- ***root***
    - npm i
- ***/server*** *(requires postgreSql)*
    - npm i
    - edit .env
    - npm run generate
    - npm run migrate
    - npm run seed
- ***/client***
    - npm i
    - edit .env

## How to run :
- (in project root) npm run dev
